# Text Alignment Pipeline

## Технические детали

Формат для рабочего документа. Документ после отработки основного алгоритма.

- Содержит строки выравниваемых документов.
    - Для одной строки может быть несколько вариантов.
- Содержит **номера строк** исходного файла разбитого по предложениям.
- Содержит коэффициенты похожести. На их основе происходит сортировка вариантов и фильтрация.
